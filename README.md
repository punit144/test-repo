# Deployment Event Platform Monorepo

Production-ready, open-source friendly Kubernetes microservice platform showcasing advanced cloud-native tooling:

- Go microservices (Gin)
- React + Vite + TypeScript dashboard
- PostgreSQL and Redis
- OpenTelemetry hooks, Prometheus metrics, JSON logging
- Unified Helm chart with Argo Rollouts, Istio, cert-manager, Gatekeeper, Argo Notifications, Calico policies
- Argo CD GitOps manifests

See [`docs/architecture.md`](docs/architecture.md) and [`docs/deployment.md`](docs/deployment.md).
